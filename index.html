How To Set Up

Step 1

The first step is to open visual studio to create your website. You will need a folder with an Index.html for your layout designs and a styles.css for your fine tweaking on pictures and font.

Step 2

The second step is to input your code to create and finish your website. Once your html code for the website is completed it is time to upload to github

Step 3

Create a github account if you have not.

Step 4

Create a new GitHub project. When the “Create a New Repository” window appears, fill in with username.github.io. Username being your GitHub account username.

Step 5

Copy your files to the new Repository folder.

Step 6

Publish your website

Step 7

Lastly, go view your website @ username.github.io
